Word ↔ Excel SMASH!

Given an MS Word DOCX file with placeholders as template, and an Excel spreadsheet table with column headers matching the placeholders, merge them: create a new word file for each row in the table.

How to use

  1. Select Template — choose a Word .docx file with {ColumnName} placeholders.
  2. Open Excel — load a spreadsheet whose column headings match the template placeholders.
  3. Validate — confirm the template and spreadsheet column names align.
  4. Merge — choose a save location; one document is produced per row.
